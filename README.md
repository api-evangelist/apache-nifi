# Apache NiFi (apache-nifi)
Apache NiFi is a dataflow management system designed to automate the flow of data between systems. It provides a web-based user interface for designing, controlling, and monitoring data flows with real-time operational control.

## APIs

### Apache NiFi REST API
The NiFi REST API provides comprehensive endpoints for managing processors, connections, controller services, process groups, reporting tasks, provenance, system diagnostics, and flow management, along with the NiFi Registry API for versioned flows.
